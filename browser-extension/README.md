# Equal ERP WhatsApp Sender (Surat, Gujarat Edition)

Professional WhatsApp automation tool tailored for **Equal ERP** collection follow-ups.

## Key Features

- **Direct WhatsApp Web Integration**: Runs inside your logged-in WhatsApp tab (no extra login required).
- **Party-Wise Aggregation**: Automatically groups multiple bills for the same party and calculates total balance.
- **Due Aging (Days)**: Tracks and displays the oldest pending bill days for each party.
- **Spam Guard (Anti-Ban)**: Randomly adds dots (`.`, `..`, `...`) to messages to prevent detection.
- **Failed Reporting**: In-app tracking of "Sent" vs "Skipped" parties with a "Show Failed" filter.
- **Manual Party Entry**: Quickly add a recipient without re-uploading your file.
- **Multi-Template Support**: Save and switch between 3 different message templates.
- **Typeable Country Codes**: Input any country code with smart suggestions.
- **Fast Search**: Instantly filter your party list by name.

## How to Install

1.  Open Chrome or Edge and go to `chrome://extensions`.
2.  Enable **Developer mode** (top right).
3.  Click **Load unpacked**.
4.  Select the `browser-extension/` folder.
5.  Pin the extension for easy access.

## How to Use

1.  Open [web.whatsapp.com](https://web.whatsapp.com) and log in.
2.  Open the extension and upload your **Equal ERP** export (.csv, .xlsx, or .xls).
3.  Review the aggregated party list in the preview.
4.  Configure your Template, Delay, and Spam Guard settings.
5.  Click **Start Campaign**.

## Support Placeholders

Use these tags in your templates:
- `{name}`: Party name
- `{balance}`: Total balance amount
- `{days}`: Oldest bill days (aging)
- `{billno}`: List of all bill numbers
- `{bills}`: Detailed list of bills and amounts (e.g., "INV-101 (Rs.500)")
